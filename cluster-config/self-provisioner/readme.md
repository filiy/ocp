# OpenShift Self Provisiong 

We can prevent any authenticated user group from self-provisioning new projects so a proper on-boarding process is followed to request a new project.

This is just updating a cluster role binding to be empty.


* https://docs.redhat.com/en/documentation/openshift_container_platform/4.19/html/building_applications/projects#disabling-project-self-provisioning_configuring-project-creation
